Design breakdown

General rules:

  • Background color of the document is #222;
  • Text color of the document is white;
  • Text size of the document document text is 12px
  • 150% line height should be used
  • Regular text in the document is either Verdana or Arial, or Helvetica (all of them - sans-serif fonts);
  • All hypherlinks are clored as #02B9E5 and have underlines removed;
  • Some elements have a problem with vertical stretching, since all their children are "floating". The "clear fix" solution from http://perishablepress.com/new-clearfix-hack/ is advised. Necessary markup is already done in HTML

Layout (general structure)

  • BODY and HTML elements have no padding or margin
  • Main content is located in element with ID "content". Its width is 1000px, it is centered (http://help.veign.com/css-center-webpage.php) and it has black background
  • Three content columns in the main part are all marked up with a class "column". They are then positioned using "float columns" method: http://www.ozzu.com/html-tutorials/tutorial-multi-column-layout-using-css-float-part-t85704.html
  • The three columns have a set width (#intro - 230px, #main - 502px, #ads - 260px, respectively), the first two of them have a padding to the right side: 4px.
  • All of the columns have background color #222;

Information blocks

  • Content inside the columns is separated into blocks. Blocks in the left column are DIV elements with a specific class "infoblock", blocks in middle section are ARTICLE elements. Such block has to have a bottom margin of 4 pixels, a black background and padding of 8px.

Top navigation menu

  • Top navigation is a simple implementation of "CSS horizontal menus" as described in http://www.secondpicture.com/tutorials/web_design/css_ul_li_horizontal_css_menu.html
  • The menu is found in the UL element with ID "mainmenu". Its padding and margin is removed. It has no default bullets or numbering.
  • All list items in navigation menu are explicitly Tahoma (or similar) font. These list items have padding top and bottom padding of 5px, left padding 10px and right padding 20px;
  • Downward arrow is created by "background bullet" technique (http://css.maxdesign.com.au/listutorial/11.htm). The bullet is taken from http://www.usanetwork.com/_img/globalheader/menuDownArrow.png

Link lists

  • There are two lists containing hypherlinks to other pages. These are both marked up with "linklist" class. The bullet used for these lists can be found at http://www.usanetwork.com/series/whitecollar/images/arrow.png

Thumbnails

  • Portrait thumbnails are also layed out using the "float" method (the unordered list "characters" has its paddings and margins removed)
  • Each list item in "characters" list is floating and has 3px padding from all sides
  • Each image inside the caracters list is marked with a class "thumb". It is 62px wide and has no border.

Buttons

  • the "Play" button on the lefthand side
    • is 200px wide and 24px high
    • its background color is #02B9E5
    • the "play" image can be found at http://www.hylandesign.com/graphics/single_arrow_right.gif; it is placed in background (background is centered vertically and left-adjusted by 10px);
    • its text color is white, text is bold and 14px;
    • text is automatically transformed into "all caps";
  • the "Subscribe" button on the righthand side is similar, but it has no defined width and height

Header

  • The huge heading "White Collar" uses a web font named "Spinnaker" which can be found here http://www.google.com/webfonts/specimen/Spinnaker. Its text size is 50px, having the line height of 60px. Its color is #B7B7B7
  • Header element has a 4px wide margin at the bottom

Page footer

  • Footer element has centered text
  • Footer element has 1px wide border at the top
  • Text size is smaller, it is 10px

Rounded corners

  • Several elements use CSS3 "rounded corners" (using this method: http://davidwalsh.name/css-rounded-corners)
  • Thumbnail images have 7px radius
  • Infoblocks have 3px radius
  • Buttons have 4px radius
